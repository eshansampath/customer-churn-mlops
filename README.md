# Customer Churn Prediction System

## Overview
This project predicts customer churn using XGBoost with MLflow tracking and SHAP explainability.

## Features
- XGBoost model with hyperparameter tuning
- Threshold optimization (F1-score)
- SHAP explainability
- MLflow experiment tracking
- FastAPI backend
- Streamlit frontend
- Docker support
